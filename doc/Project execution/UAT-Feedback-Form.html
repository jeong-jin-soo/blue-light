👤 Tester Information

A. Applicant Flow

📋 Application Status Flow

PENDING_REVIEW→ REVISION_REQUESTED↔ PENDING_REVIEW→ PENDING_PAYMENT→ PAID→ IN_PROGRESS→ COMPLETED
1Sign Up & Login
APPLICANT
  1. Go to Sign Up page and create a new applicant account
  2. Fill in name, email, phone number, and password
  3. Log in with the new account
  4. Check dashboard loads correctly
2New Application Submission
APPLICANT
  1. Click "New Application" from the dashboard
  2. Fill in all required fields (premises info, kVA, application type)
  3. Select SLD option: Self Upload or Request LEW
  4. Upload required documents (SLD drawing, site photos, etc.)
  5. Review the application summary and submit
  6. Verify the application appears in the list with "Pending Review" status
3Application Detail & Revision
APPLICANT
  1. Open a submitted application to view details
  2. If revision is requested, check the revision message from Admin/LEW
  3. Edit the application and re-upload corrected documents
  4. Resubmit and verify status returns to "Pending Review"
4Payment (PayNow / Bank Transfer)
APPLICANT
  1. Open an application with "Pending Payment" status
  2. View the pricing breakdown (kVA-based + SLD fee if applicable)
  3. Check PayNow QR code or bank transfer details are displayed
  4. Upload payment receipt
  5. Verify status changes to "Paid" after admin confirmation
5LOA (Letter of Authorization) & Document Downloads
APPLICANT
  1. In application detail, view the LOA document
  2. Complete the digital signature for LOA
  3. Download uploaded files and generated documents (licence PDF, receipt)
  4. Verify all documents are readable and correctly generated
6AI Chatbot
APPLICANT
  1. Click the chatbot icon (bottom-right corner)
  2. Ask a question about the licence application process
  3. Verify the AI responds with relevant information
  4. Check that responses render markdown correctly (lists, bold, links)
  5. Test conversation history — close and reopen the chat
7Profile & Data Management (PDPA)
APPLICANT
  1. Navigate to Profile page
  2. Update profile information (name, phone, etc.)
  3. Test "Export My Data" — download a copy of personal data
  4. Review privacy policy and consent settings
B. SLD-Only Order (Applicant)

📐 SLD Order Status Flow

PENDING_QUOTE→ QUOTE_PROPOSED→ PENDING_PAYMENT→ PAID→ IN_PROGRESS→ SLD_UPLOADED→ COMPLETED

* Applicant may reject a quote (→ QUOTE_REJECTED) or request revision after SLD upload (→ REVISION_REQUESTED)

1Create SLD-Only Order
APPLICANT
  1. Navigate to SLD Orders section
  2. Click "New SLD Order"
  3. Fill in premises info, kVA (optional), and description
  4. Upload sketch or reference materials (optional)
  5. Submit the order request
  6. Verify order appears with "Pending Quote" status
2Accept/Reject Quote & Pay
APPLICANT
  1. Open an SLD order with "Quote Proposed" status
  2. Review the proposed price from SLD Manager
  3. Accept the quote (or reject and check status changes)
  4. If accepted, upload payment receipt
  5. Verify status progresses correctly
3SLD Delivery & Revision Request
APPLICANT
  1. Open an SLD order with "SLD Uploaded" status
  2. Download and review the SLD drawing (PDF)
  3. If satisfactory, complete the order
  4. If revision needed, submit a revision request with comments
  5. Verify chat communication with SLD Manager works
C. LEW (Licensed Electrical Worker)
1Dashboard & Application Review
LEW
  1. Log in as LEW (lew@licensekaki.sg / admin1234)
  2. View the dashboard with application statistics
  3. Browse the application list and filter/search
  4. Open an application to review details and uploaded documents
2Approve / Request Revision / Reject
LEW
  1. Open a "Pending Review" application
  2. Review all documents (SLD, site photos, SP account)
  3. Approve → verify status moves to "Pending Payment"
  4. Request Revision → enter reason, verify applicant sees it
  5. Check that status transitions are correct
D. Admin
1User Management
ADMIN
  1. Log in as Admin (admin@licensekaki.sg / admin1234)
  2. Navigate to User Management page
  3. View all users with role filters (Applicant, LEW, Admin, SLD Manager)
  4. Approve a pending LEW registration
  5. Change a user's role (e.g., assign SLD_MANAGER role)
2Application Management & Payment Confirmation
ADMIN
  1. View all applications with status filters
  2. Open an application and review all details
  3. For "Paid" application, confirm payment receipt and advance status
  4. Upload licence PDF or completion documents
  5. Mark an application as "Completed"
3Price / kVA Tier Management
ADMIN
  1. Navigate to Price Management page
  2. View existing kVA price tiers
  3. Edit a tier price and save
  4. Add a new tier or remove an existing one
  5. Verify price changes reflect in new applications
E. SLD Manager
1Dashboard & Quote Proposal
SLD MANAGER
  1. Log in as SLD Manager (sldmanager@licensekaki.sg / admin1234)
  2. View the SLD order dashboard with statistics
  3. Open a "Pending Quote" order
  4. Review applicant's description, sketch, and kVA info
  5. Propose a quote price and submit
  6. Verify order status changes to "Quote Proposed"
2SLD Drawing Upload & Completion
SLD MANAGER
  1. Open a "Paid" or "In Progress" order
  2. Upload the SLD drawing file (PDF)
  3. Verify status changes to "SLD Uploaded"
  4. Handle revision requests if applicant sends feedback
  5. Re-upload revised SLD drawing
  6. Complete the order after applicant accepts
3Chat Communication with Applicant
SLD MANAGER
  1. Open an SLD order detail page
  2. Use the chat panel to communicate with the applicant
  3. Send messages and verify real-time display
  4. Test message history persistence
